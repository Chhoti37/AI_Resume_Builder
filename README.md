# AI Resume Builder 🚀

An AI-powered application that helps users create professional resumes with the help of AI-generated summaries and suggestions.

## Tech Stack
- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js, Express, MongoDB
- **AI Integration:** OpenAI / Gemini API
- **State Management:** Redux Toolkit

## Features
- AI-based professional summary enhancement.
- Multiple resume templates.
- Real-time preview and PDF download.
- Dashboard to manage saved resumes.

## How to Run
1. Clone the repo.
2. Run `npm install` in both client and server folders.
3. Add your `.env` variables.
4. Run `npm run dev`.
